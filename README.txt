HMS WEB APP - FLASK + SQLITE

Run on Windows:
1. Extract this ZIP.
2. Open the folder.
3. Double-click run_web_app.bat

Or run manually:
python -m pip install -r requirements.txt
python app.py

Open in browser:
http://127.0.0.1:5000

Default Login:
admin / admin123
manager / manager123
reception / rec123

Features:
Login roles, dashboard, rooms, customers, bookings, inventory, room service,
billing with GST/discount, printable invoice, employees, users, reports, CSV export.

Database is created automatically at:
instance/hotel_web.db
